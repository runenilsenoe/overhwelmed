# Overhwelmed (Chrome/Edge)

Fjerner artikler på nettaviser som inneholder ord du legger inn.

## Installering
1. Åpne `chrome://extensions` (eller `edge://extensions`).
2. Slå på **Developer mode / Utviklermodus**.
3. Klikk **Load unpacked / Last inn upakket**.
4. Velg mappen med denne extensionen.

## Bruk
1. Åpne en nettavis.
2. Klikk på extension-ikonet.
3. Skriv ett ord per linje og trykk **Lagre**.
4. Last siden på nytt om du ikke ser endringen med en gang.

Tips: Du kan bruke både enkeltord og uttrykk (f.eks. `premier league`).
